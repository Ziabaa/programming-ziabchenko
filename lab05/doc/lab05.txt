Лабораторна робота №5. Циклічні конструкції

Зябченко Іван Володимирович, гр. КН-922а

Завдання: Якою мінімальною кількістю купюр можна набрати потрібну суму S за умови що в наявності є купюри
номіналом 1, 2, 5 та 10 грн.?

Основна частина:
- опис роботи основної функції: для того щоб дізнатися мінімальну кількість купюр ц заданій сумі, потрібно, 
по черзі від найбільшого номіналу, спускатися до найменшого, і кожен наступний номінал повинен бути менший
за суму поточної суми, та не превищувати його.
 - перелік вхідних даних:
	-number - дане число, позитивне ціле число (int);
	-quantity - кількість купюр, позитивне ціле число (int);
	-sum - їх поточна сума, позитивне ціле число (int);
	-nominal_money1, перший номінал, позитивне ціле число (int), дорівнює 10;
	-nominal_money1, другий номінал, позитивне ціле число (int), дорівнює 5;
	-nominal_money1, третій номінал, позитивне ціле число (int), дорівнює 2;
	-nominal_money1, четвертий номінал, позитивне ціле число (int), дорівнює 1;
 - дослідження результатів роботи програми:
	-quantity - кількість купюр. Так як всі оперуємі числа є дійсними, то й результуюча змінна є дійсною.
	 Для її пошукку потрібно ввести нововивчений невизначений цикл, у якому будуть відбуватися такі послідовні дії:
		1.Дії виконуються, до того моменту, доки поточна сума не буде дорівнювати заданій сумі.
		2.Для зміни суми потрібно, щоб виконувалася одна з умов:
			-якщо поточна сума менша, або дорівнює першому номіналу, то виконати операцію додавання цього
			 номіналу, та збільшити кількість купюр на один, код виглядає так:
						 if (sum + nominal_money1 <= number) {
                        				sum += nominal_money1;
                        				quantity++; }
			-якщо поточна сума менша, або дорівнює другому номіналу, то виконати операцію додавання цього
			 номіналу, та збільшити кількість купюр на один, код виглядає так:
				                 else if (sum + nominal_money2 <= number) {
                       					 sum += nominal_money2;
                       					 quantity++; }
			-якщо поточна сума менша, або дорівнює третьому номіналу, то виконати операцію додавання цього
			 номіналу, та збільшити кількість купюр на один, код виглядає так:
				                  else if (sum + nominal_money3 <= number) {
                       					 sum += nominal_money3;
                       					 quantity++; }
			-якщо поточна сума менша, або дорівнює четвертому номіналу, то виконати операцію додавання цього
			 номіналу, та збільшити кількість купюр на один, код виглядає так:
				                  else if (sum + nominal_money4 <= number) {
                        				sum += nominal_money4;
                        				quantity++; }
		3.Тобто маємо не нескінченний цикл, який рахує мінімальну кількіть купюр у заданній сумі. Для перевірки 
		 використав компілятор, але спочатку присвоїв змінній 'number' число 28, у резільтаті, повинні мати 5 купюр:
		 10, 10, 5, 2, 1. Запустивши компілятор та зупинивши його на строці з "return 0", ввів команду "print quantity",
		 отримав резільтат '5', тобто можемо зробити висновки, що програма працює коректно.
