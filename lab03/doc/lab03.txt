1)Завдання:  За заданим опором трьох резисторів r1, r2, r3, які з’єднані паралельно, визначити загальний опір.
Основна частина:
- опис роботи основної функції: загальний опір резисторів, що з'єднані паралельно
визначаеться за формулою: r1 * r2 * r3 / r1 + r2 + r3; Де r1 - опір першого резистора,
r2 - опір другого резистора, r3 - опір третього резистора.
- перелік вхідних данних:
	resistances_1 - опір першого резистора, позитивне дійсне число (float);
	resistances_2 - опір другого резистора, позитивне дійсне число (float);
	resistances_3 - опір третього резистора, позитивне дійсне число (float);
- дослідження результатів роботи програми 
	-General_resistance - загальний опір. Так як всі оперуємі числа є дійсними, то й результуюча змінна є дійсною.
	 Тому, її тип - float - при значенні resistances_1 = 34 та resistances_2 = 23 та resistances_3 = 41, 
	 об'єм конуса повинен складати:(34 * 23 * 41) / (34 + 23 + 41) = 32062 / 98 = 327.163265
	-Для підтвердження коректності роботи програми, зупинено відлагодник на строці з "return 0" та введемо команду
	 "print general_resistance". Після вводу команди отримали наступне: 
	 (lldb) print general_resistance
 	 (float) general_resistance = 327.163269
	  Як бачимо, результат майже співпав. Похибка 0.000004 є дозволеною при виконанні операцій з плаваючою крапкою.
          Подібність результатів говорить про те, що програма працює коректно.
